Inline callout for confirmations, warnings and errors (form feedback, banners).

```jsx
<Alert variant="success" title="Message envoyé">Nous vous répondons sous 24h.</Alert>
<Alert variant="info" title="Nouvel article">Découvrez nos tendances 2026.</Alert>
```

Variants: `info` (default), `success`, `warning`, `danger` — each ships a matching icon. Props: `title`, `icon` (override).
