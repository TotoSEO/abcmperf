Icon-only button — toolbar actions, social links, carousel arrows, close buttons.

```jsx
<IconButton variant="primary" label="Suivant"><ArrowIcon /></IconButton>
<IconButton variant="outline" label="Fermer"><XIcon /></IconButton>
```

Variants: `primary`, `secondary`, `outline`, `ghost` (default). Sizes `sm|md|lg`. `square` for rounded-square. Always set `label`.
