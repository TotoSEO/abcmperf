Small uppercase status / metadata pill (Space Mono). Use for states, counts, "Nouveau", category labels.

```jsx
<Badge variant="success" dot>En ligne</Badge>
<Badge variant="accent" solid>Nouveau</Badge>
```

Variants: `neutral` (default), `accent`, `success`, `info`, `warning`, `danger`. Props: `solid` (filled), `dot` (leading status dot).
