Category / filter chip. Use for service categories, blog tags, selectable filters.

```jsx
<Tag hue="magenta">Community Management</Tag>
<Tag active onClick={...}>SEO</Tag>
<Tag onRemove={...}>Filtre</Tag>
```

Props: `hue` (logo-palette key `yellow|orange|magenta|blue|green` or any CSS colour — shows a dot), `active` (selected), `onClick` (clickable), `onRemove` (shows ×).
