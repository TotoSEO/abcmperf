Rounded surface container — the base for most content blocks.

```jsx
<Card interactive accent>
  <h3>Référencement SEO</h3>
  <p>Devenez visible sur Google.</p>
</Card>
```

Props: `padding` (`none|sm|md|lg`), `elevated` (stronger shadow), `flat` (no shadow), `interactive` (hover lift), `accent` (yellow top bar), `as`.
