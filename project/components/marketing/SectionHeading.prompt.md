Standard section opener — mono eyebrow, display title, supporting paragraph.

```jsx
<SectionHeading eyebrow="Nos services" title="Une approche 360°" description="Du site web au référencement…" align="center" size="lg" />
```

Props: `eyebrow`, `title`, `description`, `align` (`left|center`), `size` (`md|lg`), `as` (heading tag).
