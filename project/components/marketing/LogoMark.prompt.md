The ABCM lockup — circle mark + wordmark. Use in headers and footers.

```jsx
<LogoMark href="/" src="assets/logo-abcm-circles.png" />
<LogoMark markOnly />
```

Props: `src` (path to the logo PNG, relative to the page — adjust per directory depth), `markOnly`, `href`. Adapts to `.on-dark` sections automatically.
