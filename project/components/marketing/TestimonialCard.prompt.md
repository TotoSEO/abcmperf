Client review card — star rating, quote, and an author row with avatar + source. Uses the system `Avatar` for the fallback initials.

```jsx
<TestimonialCard rating={5} author="Kevin Kanel" source="Google · Avis vérifié"
  quote="Nous sommes extrêmement satisfaits du résultat." />
```

Props: `quote` (or children), `author`, `source`, `avatar` (URL), `rating` (0–5).
